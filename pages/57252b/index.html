    fiddler安装&抓包

    # Fiddler的安装、配置和手机抓包

    # 1.下载安装

    fiddler官网：Fiddler官网 (opens new window)

    安装时需要在电脑上先安装.NetFrameWork，但目前的win系统一般自带.NetFrameWork。

    # 2.Fiddler配置

    安装之后直接双击Fiddler启动应用程序，然后进行配置

    # 步骤：

    # 1.Fiddler——>Tools——>Options

    # 2.Options——>HTTPS，配置HTTPS抓包，会有证书确认安装信息弹出，确认即可。

    # 3.将证书导出，HTTPS——>Actions——>Export Root Certificate to Desktop。

    导出证书之后将会出现提示证书已经到处在桌面上的提示信息。

    # 4.Fiddler——>Tools——>Options——>Connections，配置Fiddler代理

    到这里，Fiddler配置就已经结束了。

    # 5.重启Fiddler之前，要将桌面的证书进行系统安装。桌面双击证书，进行安装。

    确认之后，将会弹出安装成功的提示信息。 接下来，重启Fiddler即可。

    # 3.测试手机抓包

    # - 先在CMD中输入ipconfig，查看电脑的IP
    # - 在手机的浏览器中输入：电脑IP+端口号
    # - 进入证书下载界面，点击Fiddler Root CERT，下载证书。

    # - 下载证书之后，进行证书安装，在设置——>安全设置——>凭据储存——>从存储设备安装，安装最新的证书即可。

    # - 在手机和电脑连接相同的WIFI网络下，在电脑命令行中敲下ipconfig，查看自己电脑的IP地址。

    # - 然后在手机的WIFI设置的代理中，选择手动设置代理，输入主机名和端口号即可。

    # - 在手机浏览器中和输入电脑的IP+端口号，进行浏览，看Fiddler是否可以进行抓包。

    如上如所示，在Host中显示的有手机请求的地址并且传回显示的协议中含有HTTP/HTTPS即可。（要保证含有HTTPS）

    以上就是Fiddler的安装、配置以及手机抓包的过程了。
