    # 选择框脚本

    • add(newOption, relOption)
    • multiple
    • options
    • remove(index)
    • selectedIndex
    • size

    选择框的value属性取值规则：

    • 如果没有选中，value为空串
    • 如果有一个选中，value等于该项的value
    • 如果有一个选中，如果该项没有value，则等于该项的文本
    • 如果有多个选择，则value等于第一个选中项的值

    HTMLOptionElement对象添加了如下属性:

    • index
    • lable
    • selected
    • text
    • value

    选择选项

    单选的话使用selectIndex属性即可，多选需要设置selected = true

    添加选项

    • appendChild()动态创建选项
    • Option够赞函数新建
    • 使用选择框的add()方法

    移除选项

    • removeChild()动态移除选项
    • 选择框的remove(index)方法
    • 将相应项设置为null

    移动和重排选项

    最好是使用DOM的插入方式

    # 表单序列化

    浏览器如何将数据发给服务器

    • 对表单字段的名称和值进行URL编码，使用&分割
    • 不发送禁用的表单字段
    • 只发送勾选的复选框和单选按钮
    • 不发送type为“reset”和“button”的按钮
    • 多选选择框中的每一个选中的值单独一个条目
    • 单击提交按钮提交表单时，也会发送提交按钮，否则不发送，也包括type为image的元素

    # 富文本编辑

    本质上就是页面中嵌入一个HTML页面的iframe。通过设置designMode属性达到编辑的目的，实际上是编辑的页面的HTML代码。需要等到onload事件才能设置designMode属性。

    使用contenteditable属性

    为元素设置contenteditable属性既可以编辑，可以不用iframe，空白页和js

    操作富文本

    与富文本编辑器交互的主要方式，是使用document.execCommand()方法。可以对文档执行预定义的命令。三个参数：

    • 执行的命令名称
    • 是否应该为为当前命令提供公户界面，兼容性的话最好设为false
    • 执行命令必须的一个值，不需要传null

    支持较好的常用命令：

    命令 值（第三个参数） 说明
    backcolor 颜色字符串 设置文档的背景颜色
    bold null 将选择的文本转为粗体
    copy null 将选择的文本复制到剪贴板
    createlink URL字符串 将选择的文本转换成一个连接，指向指定URL
    cut null 将选择的文本剪切到剪贴板
    delete null 删除选择的文本
    fontname 字符名称 将选择的文本修改为指定字体
    fontsize 1-7 将选择的文本修改为指定字体大小
    forecolor 颜色字符串 将选择的文本修改为指定的颜色
    formblock 要包围当前文本块的HTML标签 使用指定的HTML标签还格式化选择的文本块
    indent null 缩进文本
    inserthorizontalrule null 在插入字符处插入一个
    元素
    insertimage 图像URL 在插入字符处插入一个图像
    insertorderedlist null 在插入字符处插入一个
      元素
    insetunorderedlist null 在插入字符处插入一个
      元素
    insertparagraph null 在插入字符处插入一个

    元素

    italic null 将选择的文本转换成斜体
    justifycenter null 将插入光标所在的文本块居中对齐
    justifyleft null 将插入光标所在的文本块左对齐
    outdent null 凸排文本（减少缩进）
    paste null 将剪贴板中的文本粘贴到选择的文本
    removeformat null 移除掺入光标所在文本块的块级格式
    selectall null 选择文档中的所有文本
    underline null 为选择的文本添加下划线
    unlink null 移除文本的链接

    富文本选区

    使用框架的getSelection()可以确定实际选择的文本。会返回一个Selection对象，又有一下属性：

    • anchorNode
    • anchorOffset
    • focusNode
    • focusOffset
    • isCollapsed
    • rangeCount

    还有以下方法：

    • addRange(range)
    • collapse(node, offset)
    • collapseToEnd()
    • collapseToStart()
    • containsNode(node)
    • deleteFromDocument()
    • extend(node, offset)
    • getRangeAt(index)
    • removeAllRanges()
    • removeRange(range)
    • selectAllChildren(node)
    • toString()

    表单与富文本

    由于富文本编辑器使用iframe非表单，所以需要隐藏一个表单去提交。可以通过innerHTML属性去取得HTML然后操作。
