    迭代方法

    每个方法都有2个参数：

    1. 要在每一项上运行的函数
    2. 运行该函数的作用于对象（可选）

    传入这些方法中的函数都是3个参数：

    1. 数组项的值
    2. 该项在数组中的位置
    3. 数组对象本身
    • every()：对数组中的每一项运行给定函数，如果该函数对每一项都返回true,则返回true
    • filter()：对数组中的每一项运行给定函数，返回改函数会返回true的项组成的数组
    • forEach()：对数组中的每一项运行给定函数,没有返回值
    • map()：对数组中的每一项运行给定函数,返回每次函数调用的结果组成的数组
    • some()：对数组中的每一项运行给定函数,如果该函数对任一项返回true，则返回true
    var a = [1, 2, 3];
var b = a.map(function(item, index, array){
	return item * 2;
});
alert(b); //2，4，6

	return function(o1, o2){
		var v1 = o1[p];
		var v2 = o2[p];
		
		if(v1 < v2){
			return -1;
		}else if(v1 > v2){
			return 1;
		}else{

	}

    函数内部属性：

    1. arguments：函数参数类数组对象
    2. this：跟java/c#类似，引用的是函数以执行的环境对象，全局调用就是window对象
    3. caller：ES5加入，表示调用该函数的函数引用，全局下值为null
    function a(num){
	if(num <= 1){
		return 1;
	}else{
		return num * a(--num);
	}
}

//以上函数因为内部调用了自己的函数名，产生了耦合。可以用arguments.callee来解耦
function a(num){
	if(num <= 1){
		return 1;
	}else{
		//callee是一个指针，拥有一个指向拥有这个arguments对象的函数
		//ES5还加入了arguments.caller用来区别caller，严格模式会访问都会出错，为了安全
		return num * arguments.callee(--num);
	}
}
	function a( ){
		b();
	}
	
	function b( ){
		//严格模式不能给caller属性赋值，为了安全
		alert(arguments.callee.caller);
	}
	
	a(); //会弹出来a的函数体

    每个函数都包含2个属性：

    1. length：函数希望接受的命名参数的个数
    2. prototype：保存ES引用类型所哟㔹方法的真正所在，prototype属性不可枚举

    每个函数都包含非继承而来的方法（作用都是设置函数体内this对象值）：

    1. apply()：函数作用域，参数组
    2. call()：含住作用域，逐个参数
    3. bind()：ES5引入，创建一个函数实体，this值绑定传给bind()函数的值

    严格模式下this不会转成window对象，需要明确吧函数添加到某个对象或者调用apply()或者call()方法，否则就是undefined.

    window.name = "window";

//字符串替换可以用函数处理
function htmlEscape(t){
	return t.replace(/[<>"&]/g, function(match, pos, originalText){
		switch(match){
			case "<":
				return "&lt;";
			case ">":
				return "&gt;";
			case "&":
				return "&amp;";
			case "\"":
				return "&quot;";			
		}
	});
}

    Global对象

    特别对象，兜底用，你看不到它。。。

    全局作用域中定义的属性和函数都是Global对象的属性。（只要直接可以用的方法，都是的。。比如isNan()、parseInt()等）

    URI编解码：

    • encodeURI()：用于整体URI（只对普通字符编码，比如空格呀）
    • encodeURIComponent()：用于部分URI（对特殊字符编码，如：冒号、正斜杠、问号、井号），更常用
    • decodeURI()： encodeURI反操作
    • decodeURIComponent()： encodeURIComponent反操作

    以上4种方法是为了替代ES3中就开启的escape()和unescape()方法的，前者可以编码所有的Unicode字符，后者只能编码ASCII字符

    属性 说明
    Math.E 自然对数的底数，常量e
    Math.LN10 10的自然对数
    Math.LN2 2的自然对数
    Math.LOG2E 以2为底e的对数
    Math.LOG10E 以10为底e的对数
    Math.PI π的值
    Math.SQRT1_2 1/2的平方根
    Math.SQRT2 2的平方根

var max = Math.max(1, 2, 3, 4, 5);
